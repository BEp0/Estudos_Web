Este documento é para exercitar tudo que aprendi até agora...

Este é um exemplo de parágrafo, pode ser frito com a tag <p>.

Já, a tag <hr> faz essas linhas que tu estas vendo em cima e embaixo.

Essas marcações são feitas com a tag <mark>

Podemos adicionar imagens com a tag <img>

E podemos colocar de 3 formas as imagens

  • O primeiro é na mesma página do meu arquivo... Apenas colocando no src o nome do arquivo...

  • Outra forma é pegando uma imagem de uma pasta... Tem que colocar o nome da pasta antes do nome do arquivo

  • A ultima forma é pegando da WEB, usando a URL no src
